# 🌐 Network Monitoring Dashboard

A real-time **Network Monitoring Dashboard** built using **Python, Flask, and Chart.js**.  
This project monitors network connectivity, bandwidth usage, and latency in real time.

---

## 🚀 Features

- 📡 Network Connectivity Check
- ⚡ Real-time Bandwidth Monitoring
- ⏱ Latency (Ping) Monitoring
- 📊 Live Network Traffic Graph
- 🌙 Modern Dark Dashboard UI
- 🔄 Auto refresh every 2 seconds

---

## 🛠 Technologies Used

- Python
- Flask
- psutil
- ping3
- HTML
- CSS
- JavaScript
- Chart.js
